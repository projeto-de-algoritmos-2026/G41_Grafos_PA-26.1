## Sobre 
Este projeto trata-se de um jogo eletrônico em que os jogadores devem eliminar todos os quadrados, que possuem sentido de remoção.
É utilizado ordenação topológica para que o resolvedor automático funcione, uma vez que o jogador fique preso no mapa.
